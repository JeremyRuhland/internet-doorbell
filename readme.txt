Internet enabled doorbell

This atmega based board can control an RGB LED, detect a button press, switch an external load and send messages over the internet.
It is designed to be connected to an ENC28J60 based ethernet module such as this: http://www.geeetech.com/enc28j60-ethernet-module-p-263.html
Using the library found here: http://github.com/jcw/ethercard
